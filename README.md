# File Compression Tool

This project implements file compression algorithms such as RLE and Huffman coding.

## Directory Structure

- `main.cpp`: Entry point for the file compression tool.
- `file_io.h` and `file_io.cpp`: Handle file input and output operations.
- `rle_compressor.h` and `rle_compressor.cpp`: Implement RLE compression algorithm.
- `huffman_compressor.h` and `huffman_compressor.cpp`: Implement Huffman compression algorithm.
- `CMakeLists.txt`: Build configuration for CMake.
- `Makefile`: Makefile for building the project.
- `.gitignore`: Git ignore file.
